# Boundless Blocks

Boundless Blocks allows players to craft infinite versions of building blocks that do not deplete when placed. This mod is designed for creative building and large-scale projects, removing the need for repetitive material gathering.

## Features

* **Infinite Block Crafting:** Combine 9 full stacks (576 blocks total) into a single infinite block.
* **Mod Compatibility:** Compatible with blocks from other mods, including Create, Biomes O' Plenty, and more.
* **Configuration:** Includes a filtering system to allow or blacklist specific blocks via keywords.
* **Unpacking:** Infinite blocks can be converted back into 9 regular blocks at any time.
* **XP Support:** Integration with JobsAddon and LevelZ allows players to gain builder XP when placing infinite blocks.

## Requirements

### Mandatory

* **Minecraft:** 1.20.1
* **Fabric Loader:** 0.15.0 or higher
* **Fabric API**

### Optional

* **Mod Menu:** For in-game configuration access.
* **Cloth Config:** Required for the visual configuration editor.


## Usage

### Crafting Infinite Blocks

1. Collect 9 full stacks (64 items per stack) of the desired building block.
2. Open a crafting table and fill all 9 slots with the stacks.
3. Collect the resulting infinite block (indicated by the enchantment glint).
4. The block can now be placed indefinitely without being consumed from the inventory.

### Reverting Blocks

To convert an infinite block back into standard blocks, place the infinite block into a crafting grid to receive 9 regular stacks.
